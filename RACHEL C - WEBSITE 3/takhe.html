Takahe - Finding and 'Extinct' New Zealand Bird

I completed a project about the takahe bird, for this project I was given the necessary text and a few images that I could use. I had to put all this information into a website to tell the story of the 'Extinct' New Zealand bird.

This website could be used by schools to help them teach kids a history lesson. It could also be used by other people of all ages who are interested to find out a little bit more about the Takahe bird. I decided to make this site very illustrated because this is something I had never really done before and I thought it would help tell the story a little better because my illustrations would help visualise the location better.
